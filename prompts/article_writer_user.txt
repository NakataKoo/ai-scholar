# 役割
以下の情報をもとに、論文の{section}を分かりやすく解説する記事を作成してください。

# 論文解析情報
{analysis}

# 文脈（これまでのセクション）
{context}

# 内容ルール
- 網羅的に説明してください
- 重要な内容を含めてください
- 200文字以上で出力してください
- 機械学習の入門を理解できる人向けに説明してください
- 論文に記載されている事実に基づいて記述してください

# 文体ルール
- 「です・ます調」で出力してください
- 句構造文法の枠組みを用いて、理解しやすい文章を作成してください
- なるべく平易な言葉を用いて、自然な文章を出力してください
- 日本語で出力してください
- 初心者でも理解できるように嚙み砕いて説明してください

# 禁止事項
- 箇条書きを多用しないでください
- 見出し(#, ##, ###, ####)を出力に含めないでください
- 単に和訳した文章を出力しないでください
- 推測によって事実と異なる文章を書かないでください
- 3文以上にわたって同じ語尾で続けないでください（例：～ました。～ました。～ました。）
- Markdown記法を使用しないでください
