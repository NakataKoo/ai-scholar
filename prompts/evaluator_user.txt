# 評価対象記事
{article}

# 論文解析情報（参照用）
{analysis}

# 評価基準
1. 正確性：論文の内容を正確に反映しているか
2. 理解しやすさ：初心者でも理解できるよう分かりやすく書かれているか
3. 網羅性：重要な情報が漏れなく含まれているか（200文字以上）
4. 文体：「です・ます調」で自然な日本語か
5. 禁止事項チェック：
   - 箇条書きを多用していないか
   - 見出し(#)を含んでいないか
   - 単なる和訳になっていないか
   - 3文以上同じ語尾が続いていないか
   - Markdown記法を使用していないか

# 出力要件
評価結果を以下の構造で返してください：
- pass_evaluation: 合否判定（合格ならtrue、不合格ならfalse）
- feedback: 改善が必要な場合は具体的なフィードバック（合格の場合は空文字列）
- score: 各評価項目のスコア（0-10の整数）
  - accuracy: 正確性
  - readability: 理解しやすさ
  - completeness: 網羅性
  - style: 文体
  - compliance: 禁止事項遵守
